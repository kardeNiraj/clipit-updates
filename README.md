## About

This repository hosts the public update feed and release artifacts for ClipIt, a lightweight and elegant clipboard manager for macOS.

ClipIt helps you instantly access your clipboard history, recover copied content, and streamline your workflow without interrupting your focus.

This repository powers:
- Sparkle app updates
- Public release downloads
- Versioned update archives
- Delta update files

## Installation

1. Download the latest release
2. Extract the ZIP archive
3. Move ClipIt to the Applications folder
4. Launch the app

---

## Automatic Updates

ClipIt uses the Sparkle framework for seamless application updates.

When a new release becomes available, the app can automatically:
- Check for updates
- Download the latest release
- Install updates securely

## Privacy

ClipIt stores clipboard history locally on your device and does not upload clipboard data to external servers.
